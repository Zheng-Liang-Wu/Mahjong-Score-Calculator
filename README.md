# 🀄 麻將台數計算機 (Mahjong Score Calculator)

### 💡 開發動機
本專案源於家庭團聚時的實際需求。由於麻將台數計算規則繁瑣（如：莊家連莊倍率、自摸規則等），為了加速牌局進行並減少計算爭議，我自發性地將這些邏輯轉化為自動化網頁工具。

### 🛠️ 技術實作
* **前端架構**：使用 **HTML5** 與 **CSS3** 建構響應式介面，適配行動裝置使用。 
* **核心邏輯**：使用 **JavaScript** 實作：
    * **條件式渲染**：根據使用者選擇的「有無自摸」，動態切換莊家選擇與輸家下拉選單。
    * **精確算法**：處理基本底數、每台倍率以及莊家連莊的額外加乘邏輯。
    * **金額累積**：設計狀態儲存機制，即時更新每位玩家在該將牌局中的總金額變化。

### 🌟 專案亮點
* **客製化設計**：針對特定使用者（東林、奇岳、政亮、梅子）預設名稱，提升使用體驗。
* **邏輯嚴密**：區分莊家自摸與閒家自摸的不同扣款邏輯，確保金額分配正確。
* **一鍵重置**：整合清除功能，方便使用者快速開啟新回合。

---

### 🚀 如何執行
1.  將本儲存庫 Clone 到本地端。
2.  在 **VS Code** 中開啟資料夾。
3.  安裝並使用 **Live Server** 插件啟動 `Calculator.html`。
4.  或直接雙擊 `Calculator.html` 透過預設瀏覽器開啟。
